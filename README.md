# MiniSocialMediaAPI

**MiniSocialMediaAPI** es una API en desarrollo, inspirada en redes sociales como Facebook e Instagram.  
Está diseñada para aprender y reforzar conocimientos en **backend con ASP.NET Core**, implementando funcionalidades básicas de una red social.

## Estado del Proyecto
Actualmente, la API mantiene una **arquitectura básica**, pero está en proceso de evolucionar hacia una **arquitectura más limpia y escalable**.  
Este proyecto se centra principalmente en el aprendizaje y la práctica de buenas prácticas en desarrollo backend.

## Características (en desarrollo)
- Gestión de usuarios
- Publicaciones (posts)
- Comentarios y likes
- Seguimiento de usuarios
- Autenticación y autorización básica

## Objetivo
El proyecto tiene como propósito:
- Mejorar las habilidades en **ASP.NET Core**.
- Practicar la **organización de proyectos** y patrones de diseño.
- Crear una base sobre la cual se puedan implementar nuevas funcionalidades de manera más profesional en el futuro.

## Tecnologías
- **ASP.NET Core**
- Entity Framework Core
- SQL Server / Base de datos relacional
- AutoMapper

## Próximos Pasos
- Migrar a una **arquitectura limpia (Clean Architecture)**
- Añadir pruebas unitarias
- Mejorar la seguridad y autenticación
- Implementar funcionalidades avanzadas de red social
